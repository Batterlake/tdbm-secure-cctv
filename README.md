# tdbm-secure-cctv

# Запуск проекта

 Сервис: 
  1. cd deploy
  2. docker-compose build
  3. docker-compose up
  
 База данных:
  1. cd db
  2. docker-compose build
  3. docker-compose up
  
 frontend:
  1. cd fromtent
  2. pip install flask psycopg2-binary numpy opencv-python requests
  3. flask run
